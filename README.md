# CalangoFlux — Early Access MVP

> Fluxo Vivo, Impacto Real: Conectando Pessoas, Dados e Regeneração.

A primeira plataforma brasileira de automação agentic, focada em IA autônoma, Web3 e tecnologias regenerativas.

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + Framer Motion
- Supabase (PostgreSQL)
- OpenAI GPT-4 (via edge function)
- EmailJS
- Vercel (deploy)

## Desenvolvimento

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
# Deploy via Vercel CLI ou GitHub integration
```
